## Sample Questions

### What is DevOps?
DevOps is a practice that combines development and operations to automate 
software delivery and improve reliability.

### What is CI/CD?
CI/CD automates code integration, testing, and deployment to deliver software faster.

### What is Docker?
Docker is a containerization tool that packages applications with their dependencies.

### What is Terraform?
Terraform is an Infrastructure as Code tool used to create and manage cloud resources.

### What is AWS EC2?
EC2 provides virtual servers in the AWS cloud to run applications.
